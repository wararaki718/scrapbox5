# typer commandline application

## setup

```shell
pip install typer rich pytest
```

## run

```shell
python app/main.py hello tmp
```

```shell
python app/main.py bye tmptmp
```

```shell
python app/main.py bye tmptmp --formal
```

## help

```shell
python app/main.py --help
```

## test

```shell
pytest
```
